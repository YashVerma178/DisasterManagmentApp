🌍 Disaster Management App

A real-time disaster monitoring and emergency response platform designed to provide early warnings, interactive hazard mapping, and emergency assistance using AI-powered insights and GIS technology.

Built with the help of Claude AI to enhance planning, architecture, and feature development.

🚀 Features
📡 Real-Time Hazard Monitoring
-Real-time disaster data fetched from satellite & external APIs
-Multi-hazard monitoring:
   -Floods
   -Earthquakes  
   -Cyclones
   -Landslides (Expandable)
-Dynamic heatmaps for affected areas
-Hazard layering for better visualization

⚠️ Early Warning & Smart Alerts
-Location-based push alerts
-Context-aware emergency notifications
-Multi-channel alert system:
   -SMS alerts
   -App notifications
   -Voice alerts (Automated calls)

🎨 Color-Coded Risk Levels
-🔴 Red — High Risk / Immediate Danger
-🟠 Orange — Moderate Risk
-🟡 Yellow — Watch / Stay Alert
-🟢 Green — Safe Zone

🗺️ GIS-Based Interactive Map System
-Interactive disaster mapping
-Built using GIS tools like:
   -QGIS
   -ArcGIS
-Real-time route optimization
-Blockage detection
-Offline maps for low-network areas

🆘 SOS & Emergency Communication
-One-tap SOS alert
-Share live location
-Contact nearby emergency services
-Emergency broadcast system

👥 Crowd-Sourced Reporting
-Users can report:
   -Flooded roads
   -Blocked routes
   -Damaged infrastructure
-Community-driven real-time updates

🏠 Shelter Mapping & Evacuation Planning
-Nearest shelter detection
-Evacuation route planning
-Relief camp tracking
-Safe zone recommendations
